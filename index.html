About Python + JS vs Python

You already know programming ideas. Python writes them differently.

About Python

Python is the default language for a lot of AI work.

It is readable, practical, and surrounded by libraries that make data and machine learning much easier to work with.

pandas NumPy matplotlib seaborn scikit-learn
Today is not about mastering Python. It is about becoming comfortable enough to continue.
JS vs Python

Code blocks

JavaScript uses curly braces. Python uses colons and indentation.

JavaScript
if (score >= 60) {
  console.log("Pass");
}
Python
if score >= 60:
    print("Pass")
JS vs Python

Variable declaration

Python does not use let, const, or var.

JavaScript
let name = "Ravi";
const score = 82;
var passed = true;
Python
name = "Ravi"
score = 82
passed = True
JS vs Python

Printing output

print() is Python’s version of console.log().

JavaScript
console.log("Hello SPAI!");
Python
print("Hello SPAI!")
JS vs Python

Booleans and missing values

Python capitalises boolean values and uses None for empty or missing values.

JavaScript
let passed = true;
let remarks = null;
Python
passed = True
remarks = None

Python Basics

The small pieces we need before data and ML.

Python Basics

Data types

Python values can be text, numbers, booleans, or empty values.

Python
name = "Aisha"      # string
score = 88          # integer
price = 2.50        # float
passed = True       # boolean
remarks = None      # no value yet

print(name, score, passed)
Python Basics

Input is text by default

If you want to calculate with user input, convert it first.

Python
score_text = input("Enter your score: ")
score = int(score_text)

print(score + 5)
Python Logic

Conditionals

Conditionals let your program make decisions.

Python
score = 72

if score >= 80:
    print("Excellent")
elif score >= 60:
    print("Pass")
else:
    print("Try again")
Try it: change the score. Predict the output before running.
Python Logic

For loops

Use loops when you want to repeat code.

JavaScript
for (let i = 0; i < 5; i++) {
  console.log(i);
}
Python
for i in range(5):
    print(i)
Python Data Structures

Lists

Lists store multiple values in order. This matters because data often comes in groups.

Python
scores = [72, 85, 60, 91]

print(scores[0])       # first item
print(len(scores))     # number of items

scores.append(78)
print(scores)
Python Data Structures

Dictionaries

Dictionaries store labelled information using key-value pairs.

Python
student = {
    "name": "Aisha",
    "course": "DAAA",
    "score": 88
}

print(student["name"])
print(student["score"])
Python Organisation

Functions

Functions let us package repeated logic into reusable blocks.

Python
def calculate_average(numbers):
    total = sum(numbers)
    average = total / len(numbers)
    return average

scores = [72, 85, 60, 91]
print(calculate_average(scores))

Solution

Completed code

solution.py
scores = [72, 85, 60, 91, 45]

average_score = sum(scores) / len(scores)

print("Average score:", average_score)

if average_score >= 60:
    print("Class passed overall")
else:
    print("Class needs improvement")
